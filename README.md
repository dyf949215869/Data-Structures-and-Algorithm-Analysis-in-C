# Data-Structures-and-Algorithm-Analysis-in-C

## 第一章 引论
### 1.2 数学知识复习

1.2.1 指数
- X^A * X^B = X^A+B
- X^A / X^B = X^A-B
- (X^A)^B = X^(A*B)
- X^N + X^N = 2*X^N
- 2^N + 2^N = 2^(N+1)

1.2.2 对数

定理 1.1：logA B = logC B / logC A; C > 0

证明：
令 X = logC B, Y = logC A, Z = logA B; 因此C^X = B, C^Y = A, A^Z = B
